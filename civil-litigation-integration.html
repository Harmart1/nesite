Civil Litigation Page Implementation Guide

This document explains how to integrate the section content from civil-litigation-sections.html into your main civil-litigation.html page.

Step 1: Update the civil-litigation.html file

Insert the sections from civil-litigation-sections.html into your main civil-litigation.html file, right after the Services section and before the bottom navigation:

<!-- Services Section -->
<section id="services" class="page-section">
    <!-- ...existing services section content... -->
    
    <div class="continue-reading">
        <button class="continue-button" aria-label="Continue to Our Approach section" data-target="#approach">
            <span>Discover Our Approach</span>
            <svg class="arrow" xmlns="http://www.w3.org/2000/svg" viewBox="0 0 24 24" fill="currentColor">
                <path d="M7.41 8.59L12 13.17l4.59-4.58L18 10l-6 6-6-6 1.41-1.41z"/>
            </svg>
        </button>
    </div>
</section>

<!-- Insert all content from civil-litigation-sections.html here -->

<!-- Bottom navigation between practice areas -->
<div class="container">
    <div class="page-navigation">
        <!-- ...existing navigation... -->
    </div>
</div>

Note: Make sure all section IDs (#overview, #services, #approach, #areas, #process, #faq) match the href attributes in your jump navigation links.

Step 2: Verify JavaScript is Working

Ensure the JavaScript at the bottom of the page includes proper event listeners for:

  • Progress bar functionality
  • Continue reading buttons
  • Jump navigation active state

The code should already be included in your civil-litigation.html file.

Step 3: Testing the Page

After integration, test the following functionality:

  • Click each navigation link in the jump nav to ensure it scrolls to the correct section
  • Scroll the page and verify that the active state in the jump nav updates correctly
  • Click each "Continue Reading" button to ensure smooth scrolling to the next section
  • Check that the progress bar at the top of the page updates as you scroll

Applying This Pattern to Other Service Pages

Use the same approach for your other service pages (business-law.html, technology-law.html, general-practice.html):

  1. Create appropriate section content for each page
  2. Use the same jump navigation structure, updating the links and sections as needed
  3. Add continue reading buttons between sections
  4. Include the same JavaScript for scroll behavior
